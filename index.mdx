<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get Movilidad up and running in minutes with our streamlined setup process.</p>
  
  <Steps>
    <Step title="Install Dependencies">
      Clone the repository and install required packages using npm.
      
      ```bash
      git clone https://github.com/Fierdimo/movilidad.git
      cd movilidad
      npm install
      ```
    </Step>
    
    <Step title="Configure Environment">
      Set up your MongoDB connection and configure the application settings.
      
      ```json settings.json
      {
        "private": {
          "admin": {
            "user": "admin",
            "password": "your-secure-password"
          }
        }
      }
      ```
      
      <Note>The admin user is automatically created on first startup if it doesn't exist.</Note>
    </Step>
    
    <Step title="Launch the Application">
      Start the Meteor development server and access the application.
      
      ```bash
      npm start
      ```
      
      The application will be available at `http://localhost:3000`
    </Step>
    
    <Step title="Access the Dashboard">
      Log in with your admin credentials to access the management dashboard and begin managing service records.
    </Step>
  </Steps>
</div>
